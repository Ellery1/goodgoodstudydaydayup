# 贡献指南

## 文件命名规范
- 人物文件夹：使用拼音命名，如 `wang-yangming`
- 标签文件：使用中文命名，如 `困境中的自我安顿.md`
- 真题文件：格式为 `年份-地区-题目.md`，如 `2024-上海-这不过是个开场.md`

## 内容格式要求
- 使用标准Markdown语法
- 标题层级清晰（# 一级标题，## 二级标题）
- 列表项使用 `-` 或 `*`
- 代码块使用 ``` 包裹

## 新增人物流程
1. 在 `figures/` 下创建人物文件夹
2. 按模板编写 `profile.md`
3. 标注冷门指数和适配标签
4. 更新对应标签的索引文件

## 新增标签流程
1. 明确标签内涵与适用题目方向
2. 从现有 figures/ 中筛选匹配人物
3. 在 tags/ 中新建文件，列出人物及简要示例

## 质量要求
- 历史事实准确无误
- 语言符合初中生理解水平
- 素材具有启发性和实用性
- 遵循"丝滑结合"原则
