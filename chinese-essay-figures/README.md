# 初中作文素材库：历史人物×标签×丝滑结合

## 项目简介
这是一个为初中生设计的作文素材知识库，通过历史人物与标签系统的结合，帮助学生实现"丝滑结合"的写作方法。

## 目录结构
```
chinese-essay-figures/
├── README.md                 # 项目说明、使用指南、标签体系说明
├── figures/                  # 人物库（按人物维度组织）
│   ├── wang-yangming/        # 王阳明
│   ├── xu-xiake/             # 徐霞客
│   ├── zhang-dai/            # 张岱
│   └── ...
├── tags/                     # 标签索引（按标签维度组织）
├── topics/                   # 真题示例
├── templates/                # 转化模板（丝滑结合的通用方法）
├── examples/                 # 完整应用示例
├── scripts/                  # 辅助脚本
└── CONTRIBUTING.md           # 贡献指南
```

## 核心标签体系
- 困境中的自我安顿
- "无用"之用
- 边缘与主流
- 行动中的"知"
- 失败者的尊严
- 选择与代价
- 微小的坚持
- 孤独的创造力

## 使用方法
1. 根据作文题目确定核心主题
2. 选择合适的标签
3. 查找适配的历史人物
4. 使用"丝滑结合"方法进行写作
