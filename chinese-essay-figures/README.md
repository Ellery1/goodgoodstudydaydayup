# 中高考作文素材库：历史人物 × 真题实战

## 项目定位

一个面向中高考的作文素材知识库，收录 50 位中外历史人物的深度素材，配合 41 道真题（高中 11 道 + 初中 30 道），帮助学生在考场上快速调用有力的论据。

每个人物素材都经过精心打磨：
- 每段素材 600 字以上，足以支撑一篇 800 字作文的核心论证
- 跨文化对比（中西人物穿插），展示知识面的宽广
- 引用史书原文，增强可信度
- 结尾附"点睛句"，可直接用作作文中的金句

覆盖北京、上海、广州、深圳、杭州、成都、南京、武汉、重庆、长沙、西安、郑州、天津、合肥、济南、福州、昆明、南昌、苏州等城市。

每道真题包含：题目原文、审题方向、可用人物素材推荐。

## 人物素材结构

每个人物文件包含：
- **一句话定位**：一句话概括人物的核心价值
- **热门指数**：★ 到 ★★★★★，冷门人物用了更出彩
- **为什么选他**：适配关键词、跨文化对比空间、适合的题目类型
- **核心素材**（3 段）：每段 600+ 字，可直接用于作文论证
- **点睛句**：4-5 句可直接引用的金句

## 使用方式

1. 拿到作文题目后，到 `examples/` 中找类似的真题，查看推荐人物
2. 到 `figures/` 中打开对应人物文件，选择最契合题目的那段素材
3. 用"点睛句"作为段落的收尾或文章的升华

## 写作原则

本库的素材遵循以下写作标准：
- 细节驱动：用具体的场景和感官细节撬动论述
- 跨文化对比：每个人物都有中西方的关联参照
- 引用原典：《史记》《左传》《晋书》等史书原文增强说服力
- 千人千面：不同人物采用不同的叙事手法，避免套路化
